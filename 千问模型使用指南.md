## 2. 模型架构说明

`Qwen3-VL` 视觉模型由两个核心部件组成：

| 组件 | 角色 | 对应文件 |
|------|------|----------|
| **语言模型（大脑）** | 负责文字理解和生成 | `Qwen3VL-2B-Instruct-F16.gguf` |
| **视觉编码器 + 投影仪（眼睛）** | 负责"看懂"图片，将信息转换为语言模型可理解的格式 | `mmproj-Qwen3VL-2B-Instruct-F16.gguf` |

> ⚠️ **关键说明**：`mmproj` 仅作为"中继器"存在，其作用是将视觉编码器输出的图像特征转换为语言模型能"听懂"的 token 序列。它本身不参与任何内容生成、推理或安全策略判断。
>
> **安全限制的来源是语言模型（大脑）本身**，以 `Qwen3VL-2B-Instruct` 为例：
> - 模型训练阶段，官方在语言模型的权重中注入了内容安全对齐（Safety Alignment）；
> - 当用户输入涉及敏感话题时，"大脑"会主动拒绝回答——这是语言模型的内在行为，与 `mmproj` 无关；
> - 因此，**替换 `mmproj` 文件无法解除安全限制**，解除限制需要替换语言模型本身（如使用 `Huihui-Qwen3-VL-*` 等 abliterated 版本）。

## 6. OpenWebUI 部署

### 6.1 安装 UV（包管理器）

以 **管理员身份** 打开 PowerShell，执行：

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### 6.2 安装 Open WebUI

```powershell
# 永久安装（仅需执行一次）
uv tool install --python 3.11 open-webui

# 验证安装
uv --version
```

### 6.3 配置系统环境变量（管理员 PowerShell）

```powershell
# 将 uv 工具目录加入 PATH
[Environment]::SetEnvironmentVariable("Path", "C:\Users\xxx\.local\bin;" + [Environment]::GetEnvironmentVariable("Path", "User"), "User")

# 设置 PyPI 镜像
[Environment]::SetEnvironmentVariable("UV_INDEX_URL", "https://pypi.tuna.tsinghua.edu.cn/simple", "User")

# 设置 OpenWebUI 数据目录
[Environment]::SetEnvironmentVariable("DATA_DIR", "D:\qw\OpenWebUI\data", "User")
```

### 6.4 验证环境变量

```powershell
[Environment]::GetEnvironmentVariable("Path", "User")
[Environment]::GetEnvironmentVariable("UV_INDEX_URL", "User")
[Environment]::GetEnvironmentVariable("DATA_DIR", "User")
```

**预期输出示例**：

| 变量 | 示例值 |
|------|--------|
| `Path` | `C:\Users\xxx\.local\bin;C:\Users\xxx\.local\bin;...` |
| `UV_INDEX_URL` | `https://pypi.tuna.tsinghua.edu.cn/simple` |
| `DATA_DIR` | `D:\qw\OpenWebUI\data` |

> ⚠️ **注意**：如果返回为空，说明环境变量设置无效，需重新执行上一步。

### 6.5 启动 OpenWebUI

```powershell
# 1. 刷新当前会话的 DATA_DIR
$env:DATA_DIR = [Environment]::GetEnvironmentVariable("DATA_DIR", "User")

# 2. 启动服务
open-webui serve
```
